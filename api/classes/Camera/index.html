updateMatrixWorld​

▸ updateMatrixWorld(): Camera

用于更新一个对象的世界矩阵。 每个可视对象都有一个叫做 worldMatrix 的矩阵，表示这个对象在世界坐标系中的位置和方向。 当您修改对象的位置、旋转或缩放时，对象的 worldMatrix 矩阵会发生变化。如果您希望将这些变化反映到场景中，就需要调用 updateMatrixWorld 函数。
