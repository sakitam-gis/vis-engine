Vector4

vis-engine - v1.0.4 / Vector4

Class: Vector4

四维向量，四维向量表示的是一个有顺序的、四个为一组的数字组合，可以用来描述：

  • 一个位于四维空间中的点。
  • 一个在四维空间中的方向与长度的定义。
const a = new Vector4(0, 1, 0);

const b = new Vector4(0, 0, 0);

const d = a.distanceTo(b);
