Class: RenderBuffer

渲染缓冲区对象。 由于渲染缓冲区对象是只写的，因此它们通常用作深度和模板关联对象。

实例代码：

const renderBuffer = new RenderBuffer(renderer, {
  format: renderer.gl.DEPTH_COMPONENT16,
  width: 1920,
  height: 1080,
});

renderBuffer.bind();
renderBuffer.unbind();
