Class: Scene

场景对象，一般我们会将创建的网格渲染对象添加到场景中进行渲染

代码示例：

const scene = new Scene();

const points = new Mesh(renderer, { mode: renderer.gl.POINTS, geometry, program });

points.position.set(-1, 1, 0);

scene.add(points);
