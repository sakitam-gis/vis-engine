Class: Vector3

三维向量，三维向量表示的是一个有顺序的、三个为一组的数字组合，可以用来描述：

  • 一个位于三维空间中的点。
  • 一个在三维空间中的方向与长度的定义
const a = new Vector3(0, 1, 0);

const b = new Vector3(0, 0, 0);

const d = a.distanceTo(b);
