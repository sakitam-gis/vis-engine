normalized​

• Optional normalized: boolean

指定当整数值转换为浮点数时是否应该将整数数值归一化到特定的范围。

  • 对于 gl.BYTE 和 gl.SHORT，当设置为true时，归一化到 [-1, 1]。
  • 对于 gl.UNSIGNED_BYTE 和 gl.UNSIGNED_SHORT，当设置为true时，归一化到 [0, 1]。
  • 对于 gl.FLOAT 和 gl.HALF_FLOAT，此参数无效。
